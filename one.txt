每个角色词条必须包含以下四个区块：

3. 【服装档案】
4. 【北美短剧视觉定位】

必须包含以下全部信息，并写成连贯自然的一段描述：

- 性别、年龄段、种族/族裔视觉倾向（若剧本可判断；若不能判断则标注[待确认]）
- 身高感、体态、肩颈线条、站姿/坐姿/走路姿态特征
- 发型发色：长度、卷直程度、发量、发际线、刘海、鬓角、是否扎发、是否有层次
- 面部结构：额头、眉骨、眼窝、颧骨、鼻梁、鼻头、唇形、下颌线、下巴、面部宽窄比例
- 面部特征至少 7 个明确细节，例如：
  - 眉形
  - 眼型
  - 眼神气质
  - 鼻型
  - 唇形
  - 皮肤状态
  - 颧骨
  - 下颌线
  - 法令纹
  - 泪沟
  - 痣
  - 疤痕
  - 雀斑
  - 黑眼圈
  - 牙齿观感
- 整体脸型轮廓感
- 气场关键词：例如“脆弱但倔强”“高位压迫感”“危险而克制”“冷艳贵气”“受过苦但不服输”
- 如角色具有强“北美爆款短剧”标签，请额外写出：
  - 这是“豪门压迫型 / 高位Alpha型 / 破碎感女主型 / 甜美受欺负型 / 冷感反派型 / 控制欲未婚夫型 / 强势母亲型 / 心机闺蜜型 / 野性狼人型 / 禁欲继承人型”等哪类视觉 archetype

写法要求：

## 2）【表演基线】

该区块用于后续视频提示词中的人物表演统一。

必须包含以下内容：

### 情绪底色
用一句话概括角色在本剧中的默认情绪状态。
例如：
- “她表面克制顺从，底层一直绷着不肯认输的倔强。”
- “他默认处于高位审视状态，说话不急，但压迫感持续存在。”

### 情绪弧线
如果角色在剧中有明显变化，请用：
“起点 → 转折 → 终点”
格式概括。
例如：
“羞辱中求生 → 被逼到极限后的第一次反击 → 学会利用自己的位置反压对方。”

### 标志动作
写 2-4 个最具辨识度、会反复出现的肢体习惯。
必须是能被镜头拍到的动作，而不是抽象性格词。
例如：
- 回答前先短暂停顿半秒
- 生气时下颌收紧，舌尖顶住上颚
- 紧张时拇指反复摩挲戒指边缘
- 说狠话时不提高音量，只微微前倾压近对方
- 被羞辱时先移开视线，再强行抬眼对视

### 声音特征
用于表演提示词。
包括：
- 音色
- 音域
- 语速
- 发声位置
- 说话习惯
- 是否带口音/阶层感/教育感/支配感
例如：
“偏低沉的胸声，语速慢，句尾经常压低，不轻易提高音量，但每个重音都带控制感。”

### 情绪外化方式
说明这个角色的情绪主要通过什么被看见：
- 眼神变化为主
- 嘴角与呼吸为主
- 姿态控制为主
- 说话节奏突变为主
- 身体距离入侵为主
- 假笑与停顿为主

每套服装必须用以下格式输出：

【服装A｜适用场景】
一整段可复制的服装描述，必须包含：
- 上装
- 内搭
- 下装
- 鞋
- 包/首饰/腕表/戒指/耳饰/腰带等配饰

要求：

# 二、场景词条

为每一个独立场景建立词条。

“独立场景”定义：
- 地点变化；
- 同一地点但时间明显跳跃；
- 同一地点但空间状态明显改变；
- 同一地点但社会关系/情绪功能明显变化，也可单独拆条。

每个场景词条必须包含以下三个区块：
1. 【空间锁定】
2. 【光照状态】
3. 【北美短剧镜头价值】

---

## 1）【空间锁定】

写成一整段可直接复制的空间描述。

必须包含：
- 场景类型：室内 / 室外 / 半室外
- 空间性质推断：例如“低收入社区边缘地带”“豪门宴会厅”“私人医院VIP病房”“律师事务所高层办公室”“家族庄园走廊”“地下停车场”
- 空间尺度感：面积、层高、压迫或开阔感
- 布局与陈设：必须按照画面方位来写
  - 前景
  - 中景
  - 后景
  - 左侧
  - 右侧
  - 中央
  - 深处/尽头
- 墙面 / 地面 / 天花板：
  - 材质
  - 颜色
  - 磨损状态
- 剧本明确出现的陈设必须保留
- 剧本未明写但对画面成立必要的补充要标注[补充]
- 必须额外判断该场景是否带有：
  - 阶层符号
  - 权力符号
  - 情感压迫符号
  - 危险/秘密感
  - 欲望/诱惑感
  - 羞辱/审判感

最后请给出：
- 材质质感关键词（3-6个）
- 氛围关键词（3-6个）

要求：
- 不只写“房间里有什么”，而要写“这个空间拍起来是什么感觉”；
- 要符合北美真人短剧的空间美术逻辑，而不是空洞的文学描写。

---

## 2）【光照状态】

如果该场景在剧中有多个时间段，请分“日间 / 黄昏 / 夜间 / 雨夜 / 清晨”等子条目分别描述。

每个时段必须包含：
- 时段标识
- 主光源：类型、方向、色温
- 辅助光源：有无 practical light（台灯、壁灯、烛光、吊灯、窗外路灯、车灯、霓虹）
- 光照分布：亮区 / 暗区 / 阴影区 / 逆光区 / 光柱
- 面部受光特征：适合把人物拍得更脆弱 / 更危险 / 更高位 / 更冷酷 / 更暧昧
- 色彩基调：主色、辅色、饱和度策略
- 是否适合高对比度戏剧光
- 是否适合用来做钩子镜头 / 情绪对峙 / 身份揭露 / 偷听 / 告白 / 审判 / 反击

# 四、视觉风格锁定

这是本剧所有后续 AI 画面提示词的统一风格基准。
这一部分必须写得足够明确，后续每个镜头单元都可以直接在开头调用。

## 1）【画面比例】
默认 9:16 竖屏。

请说明：

- 画面构图应如何服务人物关系压迫、身份差、情绪推进

## 2）【影调与色彩】
请明确写出：
- 整体明度倾向
- 对比度倾向
- 暗部偏色
- 亮部偏色
- 主色调及主要载体
- 点缀色及其载体
- 饱和度策略
- 是否采用“豪门冷调”“危险暖金”“贫穷灰褐”“医院惨白”“雨夜蓝黑”“宴会高反差金黑”等类型

## 3）【质感】
请明确：
- 偏数码干净 / 轻胶片 / 微颗粒 / 奶油高光 / 硬锐利 / 哑光肤感 / 商业广告级皮肤质感
- 颗粒浓度
- 皮肤质感处理方向
- 服装与珠宝、皮革、玻璃、金属、丝绸如何表现

## 4）【镜头偏好】
必须明确：
- 运镜倾向：克制、推进、环绕、突然逼近、轻微手持、压迫式推近
- 景深偏好：浅景深 / 中景深 / 重要道具极浅景深
- 景别偏好：特写、近景、中近景是否优先
- 对话机位策略
- 情绪镜头策略
- 冲突镜头策略
- 钩子镜头策略

要求特别考虑“北美爆款短剧”的画面需求：
- 镜头优先服务脸、眼神、关系张力；
- 同框时优先制造体型差、站位差、阶层差、压迫感；
- 爆点时优先给特写，而不是大远景；
- 要让观众在手机上刷到时，1秒内看到情绪冲突。

## 5）【北美爆款短剧专属风格规则】
这是新增强化模块，必须单独写出。
请总结本剧在视觉上应优先遵守的 6-10 条规则，例如：
- 人物脸部可读性高于复杂调度
- 女主脆弱感与反抗感必须同时成立
- 男主/高位者必须具备压迫性的空间占有感
- 豪门、狼人、家族、契约、复仇类元素要有清晰权力视觉符号
- 情绪爆点优先极近景和呼吸感表演
- 同一句狠话要留反应镜头，不可只拍说话者
- 重要物件必须能拍特写并具备故事感
- 场景既要有真实质感，也要有商业化戏剧美感

## 6）【风格声明模板】
基于以上所有内容，输出一句可直接复制到每个镜头/单元开头的固定风格声明。
要求：
- 简洁

- 可直接复用

# 五、全剧情绪地图

必须包括：
- 起始情绪状态
- 叙事主体一开始处在什么压力之下
- 每一次关键情绪转向发生在什么位置、由什么事件触发
- 核心情绪支点 1-2 个，为什么它们是全剧最重要的情绪枢纽
- 情绪如何从“受压”转成“反击”、从“误解”转成“揭露”、从“吸引”转成“撕裂”或“联手”
- 最终情绪落点是什么
- 如果是多角色结构，以主角为主线，其他角色只做对位补充

要求：
- 强调“观众刷短剧时的情绪体验”，不是文学性的主题分析；
- 要写出“观众为什么会继续追”的情绪驱动力；
- 要让后续导演、剪辑、分镜人员一眼看懂本剧情绪引擎在哪里。

# 工作规则

1. 所有描述都必须“能被画出来、能被拍出来、能被AI稳定生成出来”。
   禁止空泛形容，禁止只写抽象气质不写可见细节。

2. 剧本中明确写出的信息必须忠实提取，不可擅自篡改。
   若必须补充，必须标注[补充]。

3. 一切判断以“北美爆款短剧画面效果”为优先：
   - 高识别
   - 高情绪
   - 高关系张力
   - 高权力符号
   - 高钩子可视性

4. 不要把角色写成普通路人脸，不要把场景写成普通电视剧场景。
   要明确谁更像高位者，谁更像被压迫者，谁更危险，谁更值得镜头停留。

5. 人物描述优先符合“欧美真人商业短剧”的审美体系：
   - 脸部轮廓明确
   - 情绪可读性强
   - 近景成立
   - 特写耐看
   - 关系同框有戏

6. 如果剧本包含豪门、契约、狼人、家族、继承、复仇、婚约、背叛、秘密身份、权力压制等元素，请在角色、场景、物件、风格中强化其视觉符号。

7. 输出必须完整、系统、条理清晰，不得偷懒省略。
   不要说“其余类似”“同上略”。

8. 你的输出最终目的是：让这部剧后续所有AI视觉生产都能“统一、稳定、可复用、爆点明确”。
